
--------------------------------------------------------------------------------
Directory structure for WIT-J files

Note:
   Most of these directories are in RTC. However, the following directories are
   not in RTC; they are generated by "build" scripts:

      java/bytecode
      java/bytecode/com/ibm/witj
      test/bytecode/com/ibm/witj/test
--------------------------------------------------------------------------------

Revised 1/30/13

Directory          Contents
---------          --------

witj                  
  cpp                
    linux          Makefile, Object files, and .so file for WIT-J/C++ on Linux
      dummyHeaders Dummy header files to avoid warnings from make depend
    winsdk         Makefile, Object files, and .dll file for WIT-J/C++ on
                   Windows SDK.
    src            Source files, appl.mk & sources.mk for WIT-J/C++
  doc              Documumentation
  examples
    demo1          Source, bytecode, Jar and WIT msg files for the WitJDemo1
                   example
    partlister     Source, bytecode, Jar and data files for the WitJPartLister
                   example
  java               
    bytecode       Jar file for WIT-J/Java
      com
        ibm
          witj     Bytecode files for WIT-J/Java
    src            Manifest file for the WIT-J/Java Jar file
      com
        ibm
          witj     Source files for WIT-J/Java
  scripts          Scripts for working with WIT-J
  test               
    bytecode       Jar file for the WIT-J test program.
      com 
        ibm 
          witj 
            test   Bytecode files for the WIT-J test code.
    data           Data files for the WIT-J tests.
    src
      com
        ibm
          witj
            test   Source files for the WIT-J test code.
